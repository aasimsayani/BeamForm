# BeamForm

## Instructions 
 - Fork and clone
 - 'cd beam-form'
 - 'yarn install'
 - 'yarn start'

## Goal

Automate the current process for Beam where a user uploads their information, and change the current data path.

- React 
- mySQL
- WordPress
